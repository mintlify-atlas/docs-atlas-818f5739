The `Item` class represents an item instance with an ID and quantity, while `ItemComposition` provides detailed information about item types.

## Item Class

The `Item` class is a simple value class containing:

```java
public class Item {
    private final int id;
    private final int quantity;
}
```

### Creating Items

```java
Item item = new Item(ItemID.ABYSSAL_WHIP, 1);
int id = item.getId();
int quantity = item.getQuantity();
```

### Getting Items

Items are typically obtained from item containers:

```java
ItemContainer inventory = client.getItemContainer(InventoryID.INVENTORY);
if (inventory != null) {
    Item[] items = inventory.getItems();
    for (Item item : items) {
        if (item.getId() != -1) { // -1 = empty slot
            int id = item.getId();
            int qty = item.getQuantity();
            // Process item
        }
    }
}
```

## Example: Item Scanner

```java
public void scanInventory() {
    ItemContainer inventory = client.getItemContainer(InventoryID.INVENTORY);
    if (inventory == null) return;
    
    int totalValue = 0;
    List<Item> foodItems = new ArrayList<>();
    
    for (Item item : inventory.getItems()) {
        if (item.getId() == -1) continue;
        
        ItemComposition comp = client.getItemDefinition(item.getId());
        
        // Calculate value
        int itemValue = comp.getPrice() * item.getQuantity();
        totalValue += itemValue;
        
        // Find food items
        String[] actions = comp.getInventoryActions();
        if (Arrays.asList(actions).contains("Eat")) {
            foodItems.add(item);
        }
        
        // Log item info
        log.info("Item: " + comp.getName() + 
                 " x" + item.getQuantity() +
                 " (" + itemValue + " gp)");
    }
    
    log.info("Total inventory value: " + totalValue + " gp");
    log.info("Food items: " + foodItems.size());
}
```

## Example: Item Filter

```java
public List<Item> getItemsMatching(Predicate<ItemComposition> filter) {
    ItemContainer inventory = client.getItemContainer(InventoryID.INVENTORY);
    if (inventory == null) return Collections.emptyList();
    
    List<Item> matches = new ArrayList<>();
    
    for (Item item : inventory.getItems()) {
        if (item.getId() == -1) continue;
        
        ItemComposition comp = client.getItemDefinition(item.getId());
        if (filter.test(comp)) {
            matches.add(item);
        }
    }
    
    return matches;
}

// Usage examples:
// Get all members items
List<Item> membersItems = getItemsMatching(ItemComposition::isMembers);

// Get all stackable items
List<Item> stackableItems = getItemsMatching(ItemComposition::isStackable);

// Get all tradeable items
List<Item> tradeableItems = getItemsMatching(ItemComposition::isTradeable);

// Get items worth more than 1000 gp
List<Item> expensiveItems = getItemsMatching(
    comp -> comp.getPrice() > 1000
);

// Get equipable items
List<Item> equipment = getItemsMatching(
    comp -> comp.isEquipable() != -1
);
```

## Example: Item Name Lookup

```java
public int findItemIdByName(String name) {
    // Note: This is not efficient for large searches
    // Consider caching or using a dedicated service
    
    int maxItemId = client.getItemCount();
    for (int id = 0; id < maxItemId; id++) {
        ItemComposition comp = client.getItemDefinition(id);
        if (name.equalsIgnoreCase(comp.getName())) {
            return id;
        }
    }
    return -1;
}

public List<Integer> findItemIdsByNameContains(String search) {
    List<Integer> results = new ArrayList<>();
    int maxItemId = client.getItemCount();
    
    for (int id = 0; id < maxItemId; id++) {
        ItemComposition comp = client.getItemDefinition(id);
        String name = comp.getName();
        if (name != null && name.toLowerCase().contains(search.toLowerCase())) {
            results.add(id);
        }
    }
    
    return results;
}
```
