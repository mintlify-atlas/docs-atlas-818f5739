Widgets are on-screen UI elements in RuneScape, such as the minimap, inventory interface, chatbox, and more.

## Getting Widgets

### By Group and Child ID

<ParamField path="groupId" type="int" required>
  The widget group (interface) ID
</ParamField>

<ParamField path="childId" type="int" required>
  The child widget ID
</ParamField>

<ParamField path="Returns" type="Widget" nullable>
  The widget, or null if not found
</ParamField>

```java
// Get chatbox widget
Widget chatbox = client.getWidget(InterfaceID.CHATBOX, 0);

// Get inventory container
Widget inventory = client.getWidget(InterfaceID.INVENTORY, 0);

// Get bank inventory
Widget bankInventory = client.getWidget(InterfaceID.BANK, 1);
```

### By Component ID

<ParamField path="componentId" type="int" required>
  The packed component ID
</ParamField>

<ParamField path="Returns" type="Widget" nullable>
  The widget
</ParamField>

```java
Widget widget = client.getWidget(componentId);
```

### Root Widgets

```java
Widget[] roots = client.getWidgetRoots();
for (Widget root : roots) {
    if (root != null) {
        // Process root widget
    }
}
```

## Widget Properties

### getId

<ParamField path="Returns" type="int">
  The widget ID
</ParamField>

Gets the widget's packed ID.

```java
int id = widget.getId();
```

### getType

<ParamField path="Returns" type="int">
  The widget type
</ParamField>

Gets the type of the widget.

```java
int type = widget.getType();
if (type == WidgetType.LAYER) {
    // Container widget
}
else if (type == WidgetType.TEXT) {
    // Text widget
}
else if (type == WidgetType.GRAPHIC) {
    // Image widget
}
else if (type == WidgetType.MODEL) {
    // 3D model widget
}
else if (type == WidgetType.INVENTORY) {
    // Inventory grid widget
}
```

### setType

<ParamField path="type" type="int" required>
  The widget type to set
</ParamField>

Sets the widget type.

```java
widget.setType(WidgetType.TEXT);
```

## Widget Hierarchy

### getParent

<ParamField path="Returns" type="Widget" nullable>
  The parent widget
</ParamField>

Gets the parent widget if this is a child.

```java
Widget parent = widget.getParent();
```

### getParentId

<ParamField path="Returns" type="int">
  The parent ID, or -1 if no parent
</ParamField>

Gets the ID of the parent widget.

```java
int parentId = widget.getParentId();
```

### getChildren

<ParamField path="Returns" type="Widget[]" nullable>
  Array of child widgets (sparse)
</ParamField>

Gets the dynamic children in a sparse array.

```java
Widget[] children = widget.getChildren();
if (children != null) {
    for (Widget child : children) {
        if (child != null) {
            // Process child
        }
    }
}
```

### getChild

<ParamField path="index" type="int" required>
  The child index
</ParamField>

<ParamField path="Returns" type="Widget" nullable>
  The child widget
</ParamField>

Gets a specific child widget.

```java
Widget child = widget.getChild(0);
```

### getDynamicChildren / getStaticChildren / getNestedChildren

```java
Widget[] dynamicChildren = widget.getDynamicChildren();
Widget[] staticChildren = widget.getStaticChildren();
Widget[] nestedChildren = widget.getNestedChildren();
```

## Position and Size

### Position

```java
// Relative position (to parent)
int relativeX = widget.getRelativeX();
int relativeY = widget.getRelativeY();

// Canvas position (absolute)
int canvasX = widget.getCanvasLocation().getX();
int canvasY = widget.getCanvasLocation().getY();

// Original position
int originalX = widget.getOriginalX();
int originalY = widget.getOriginalY();

// Set original position
widget.setOriginalX(100);
widget.setOriginalY(50);
widget.revalidate(); // Apply changes
```

### setForcedPosition

<ParamField path="x" type="int" required>
  X position relative to parent
</ParamField>

<ParamField path="y" type="int" required>
  Y position relative to parent
</ParamField>

Forces the widget to a specific position.

```java
widget.setForcedPosition(100, 50);
// Clear forced position
widget.setForcedPosition(-1, -1);
```

### Size

```java
int width = widget.getWidth();
int height = widget.getHeight();

// Set size
widget.setOriginalWidth(200);
widget.setOriginalHeight(100);
widget.revalidate();
```

### Bounds

```java
Rectangle bounds = widget.getBounds();
int x = bounds.getX();
int y = bounds.getY();
int width = bounds.getWidth();
int height = bounds.getHeight();
```

### Item IDs and Quantities

```java
if (widget.getType() == WidgetType.INVENTORY) {
    int[] itemIds = widget.getItemIds();
    int[] itemQuantities = widget.getItemQuantities();
    
    for (int i = 0; i < itemIds.length; i++) {
        int itemId = itemIds[i];
        int quantity = itemQuantities[i];
        
        if (itemId > 0) {
            // Slot contains an item
            itemId--; // Item IDs are stored as id+1

            String name = comp.getName();
        }
    }
}
```

## Example: Find Widget by Name

```java
public Widget findWidgetByName(String name) {
    for (Widget root : client.getWidgetRoots()) {
        if (root == null) continue;
        
        Widget found = searchWidget(root, name);
        if (found != null) {
            return found;
        }
    }
    return null;
}

private Widget searchWidget(Widget widget, String name) {
    if (name.equals(widget.getName())) {
        return widget;
    }
    
    Widget[] children = widget.getChildren();
    if (children != null) {
        for (Widget child : children) {
            if (child == null) continue;
            Widget found = searchWidget(child, name);
            if (found != null) {
                return found;
            }
        }
    }
    
    return null;
}
```
